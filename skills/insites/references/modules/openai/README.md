# pos-module-openai

OpenAI integration module for AI-powered features.

**Optional module** — install when you need AI/LLM capabilities.

## Install

```bash
insites-cli modules install openai
```

## Setup

Set your OpenAI API key:

```bash
insites-cli constants set --name OPENAI_API_KEY --value "sk-..." dev
```

## Features

- OpenAI API integration
- Chat completions
- Embeddings generation
- AI-powered content features
